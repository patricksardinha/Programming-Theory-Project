Player can move forward - back - left - right with boundaries.
Player can jump.
Blocks of rocks move from top screen to back (3 lines).
Player should run on it & not fall.
Score increase with the distance done.
Before each stage, the player choose a hidden bonus/malus which can fit more or less with the current level.
Each selectable bonus/malus are represented as colored cards which some geometric forms.
During the level before selecting the cards, somes cluds are hidden in it to do the best choice possible.
